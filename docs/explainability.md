## 📊 What the Module Explains

### SHAP (SHapley Additive exPlanations)
- **Theory-based**: Grounded in cooperative game theory
- **Explainable**: Shows feature contribution to each prediction
- **Fair**: Each feature gets proper "credit" for prediction
- **Visual**: Creates intuitive, professional charts

### Credit Risk Application
The module answers:
- "Why was this client approved/rejected?"
- "Which factors matter most for risk assessment?"
- "Are we discriminating unfairly?"
- "What should clients focus on to improve?"
