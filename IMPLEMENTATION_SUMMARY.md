## 📦 Module: src/modeling/explain.py

### File Statistics
- **Total Lines**: ~550
- **Functions Implemented**: 8 (all required + bonus)
- **Docstrings**: Full Portuguese documentation
- **Comments**: Production-grade inline comments
- **Type Hints**: Full type annotations
- **Error Handling**: Comprehensive validation

### Functions Implemented

#### 1. `load_model(model_path=None)` ✓
- **Purpose**: Load trained model from disk
- **Features**: PathLib support, file validation, logging
- **Returns**: Loaded sklearn Pipeline
- **Error Handling**: FileNotFoundError with clear message

#### 2. `load_data(features_path=None)` ✓
- **Purpose**: Load dataset from Parquet
- **Features**: Target column validation, shape logging
- **Returns**: Pandas DataFrame
- **Error Handling**: FileNotFoundError, ValueError for missing columns

#### 3. `sample_data(data, max_samples=2000, random_state=42)` ✓
- **Purpose**: Memory-efficient sampling with stratification
- **Features**: Preserves class distribution, handles edge cases
- **Returns**: Sampled DataFrame
- **Performance**: Prevents memory explosion on large datasets

#### 4. `create_explainer(model, X)` ✓
- **Purpose**: Initialize SHAP Explainer
- **Features**: Auto-detects Pipeline, tries TreeExplainer first
- **Returns**: SHAP Explainer object
- **Fallback**: KernelExplainer if TreeExplainer unavailable

#### 5. `generate_shap_values(explainer, X)` ✓
- **Purpose**: Compute SHAP values for features
- **Features**: Handles batch processing, comprehensive logging
- **Returns**: shap.Explanation object
- **Performance**: Efficient value computation

#### 6. `plot_summary(shap_values, X, output_path=None)` ✓
- **Purpose**: Generate SHAP summary visualization
- **Features**: High resolution (300 DPI), tight layout, color mapping
- **Output**: PNG file saved to reports/figures/
- **Visualization**: Shows feature impact with color gradient

#### 7. `plot_feature_importance(shap_values, output_path=None)` ✓
- **Purpose**: Bar chart of feature importance
- **Features**: Sorted by magnitude, clean layout
- **Output**: PNG file saved to reports/figures/
- **Visualization**: Mean absolute SHAP value per feature

#### 8. `main(max_samples=2000, model_path=None, features_path=None)` ✓
- **Purpose**: Orchestrate complete SHAP pipeline
- **Features**: Central error handling, step-by-step logging
- **Bonus**: Configurable parameters, custom paths
- **Effect**: Generates all visualizations end-to-end
